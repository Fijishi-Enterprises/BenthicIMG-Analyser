If possible, please compose your source's label-set with verified labels, and avoid duplicate labels. This will ensure more studies are compatible across sources. Verified labels will be updated as the scientific literature evolves.

Verified

A label is verified after it has been inspected by our label-set curator. It needs to comply with the following criteria:

  • Correct spelling.
  • Default Short Code formatting consistent with other labels.
  • Satisfactory description.
  • Not an exact duplicate of another label.

Duplicate

If a non-verified label semantically represents the same thing as a verified label, it may be marked as a duplicate.

Neither

If a label is neither verified nor duplicate it means that either

  • The label is unique but bespoke to a certain study.
  • It has not yet been inspected by the label-set curator.

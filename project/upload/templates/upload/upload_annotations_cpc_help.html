Coral Point Count (CPCe) saves point positions and annotations in .cpc files - one .cpc file per image. You can upload .cpc files here to import the points and annotations into your source.

You can select multiple files, but you can only select files within a single folder. Tip: When selecting files, click inside the folder area and then hit Ctrl+A in Windows / Cmd+A in Mac to select all files in that folder.

  • The image filenames used by the CPC files must match existing image filenames in your CoralNet source. The names of your folders do not matter.
  • When you upload a CPC file for an image, all existing points and annotations for that image will be overwritten. The preview will warn you if you are overwriting any confirmed annotations.
  • Each CPC file is allowed to contain labels for all, some, or none of the points.
  • Label codes used in the CPC file must match a label code in your source's labelset. Codes are case-insensitive. If your labelset has a label code PORIT, and the CPC file has Porit, then these codes will match.
